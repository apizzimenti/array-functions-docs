    f

    new f()

    Parent object for all funk methods. Each time a method is run, funk keeps a reference to the last array it was fed before it was modified. Constructing f (i.e. new f()) is not required. You can simply use f.<method> without instantiation.

    lastArray Array Reference to the last array modified.
    throwError function Typechecks parameters.
